Ember.GestureDelegate Class

Extends Ember.Object
Defined in: ../packages/ember-touch/lib/system/gesture_delegate.js:6
Module: ember-touch
Parent Module: ember

GestureDelegate allows GestureManager instances decide if a touch event can be dispatched to a view gesture.

Gestures can be set up using a GestureDelegate to coordinate the gesture recognition based on application logic.
