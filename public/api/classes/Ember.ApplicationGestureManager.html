Ember.ApplicationGestureManager Class

Extends Em.Object
Defined in: ../packages/ember-touch/lib/system/application_gesture_manager.js:8
Module: ember-touch
Parent Module: ember

An ApplicationGestureManager instance is registered into the container to inform GestureManager instances if touch events can be dispatched and it stores application gestures and delegates.

GestureManager instances deny dispatching events whenever the isAllBlocked property is true or isBlocked is true and the shouldReceiveTouch response is false.
