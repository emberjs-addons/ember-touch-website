Ember.ApplicationGestureManager Class

Extends Em.Object
Defined in: ../packages/ember-touch/lib/system/application_gesture_manager.js:7
Module: ember-touch
Parent Module: ember

Allow any view to block the gesture recognition.

When ApplicationGestureManager isBlocked, gestureManager will call shouldReceiveTouch method and when it returns false, it will deny passing touchEvents to view gestures.
